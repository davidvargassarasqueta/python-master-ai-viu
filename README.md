# Análisis Financiero con Python
Trabajo final para asignatura Introducción al Python de la Univerisdad Internacional de Valencia.

## Contenido del Repositorio
* **finanzas_utils.py**: Librería propia con funciones modularizadas para cálculo de retornos y volatilidad.
* **Trabajo_Final_Python.ipynb**: Notebook principal con la resolución de los ejercicios:
  1. Comparación de activos (Retornos y Volatilidad).
  2. Análisis Técnico (Golden Cross y Medias Móviles).
  3. Procesamiento de Series Temporales (Limpieza de datos y Resample).
  4. Web Scraping del S&P 500 (Wikipedia).

## Librerías utilizadas
* Pandas
* Yfinance
* Matplotlib
* Numpy
* Requests
